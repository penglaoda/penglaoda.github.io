hexo介绍

2015-11-08

A fast, simple & powerful blog framework—hexo

安装hexo之前需要安装：
node.js
git

安装

$ npm install -g hexo-cli

命令

$ hexo -init [folder] #Initializes a website
$ hexo -new [layout] <title> #新建文章
$ hexo -generate #生成静态网页，每次修改后都需要执行此命令
$ hexo -server #在本地启动服务器，http://localhost:4000/
$ hexo -deploy #部署网站到github或者其他地方

主题

下载主题后，修改config.yml文件中的theme选项

cd themes
git clone xxxxxxxxxxxxxxxx /## 粘贴上URL ##/

_config.yml文件部署设置

注意事项

  1. 每次创建（修改）博客后，都可以先在本地查看，然后上传部署到Git上
  2. 每次deploy之前，需要先执行generate生成静态文件

总结

因为 Hexo 生成的一套静态网页，我们就可以把 “public”（默认）下的文件全部放在某个免费的空间（比如七牛云存储、Github 空间）上，不需要借助任何 Web 服务器，我们就可以搭建一个基本的网站出来。对于个人博主来说，这样省时省钱，是一个很经济的方法。
