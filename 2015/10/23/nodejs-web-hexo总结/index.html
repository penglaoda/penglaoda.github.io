nodejs_web_hexo总结

2015-10-23

最近这段时间尝试用node.js做网站开发（本来想学PHP来着，被朋友推荐来学习这个，说以后会是趋势），因为博主之前没怎么接触过web开发，所以暂时还没做出来一个完整的网站，不过这个博客网站还是被我弄了出来，虽然不难，但还是小有成就感的。

本人也是新手，这只是个人笔记，不是教程，仅供参考。
以下内容难免有错漏之处，还请见谅。

在学习过程中，涉及到的知识点如下所示

  • node.js

  • hexo
  • markdown

nodejs

简单来说，javascript通常是作为前端语言运行在前端控制浏览器的，而现在javascript可以作为web服务器后端语言，而且比PHP等更加底层，比如PHP有APACHE，而node.js是自己编写，不过不用我们编，我们直接调用核心模块就行。

快速搭建node.js环境

express

基于node.js的web框架，简单点说就是把一些最基本的操作封装成核心模块，让我们可以更加方便的编写web应用，目前官方推荐的就是这个框架。

$ mkdir project && cd project
# 这里没有从官方 npm 安装，而是使用了大淘宝的 npm 镜像
$ npm install express --registry=https://registry.npm.taobao.org
